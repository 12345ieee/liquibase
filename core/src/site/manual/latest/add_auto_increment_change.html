"Add Auto-Increment" Refactoring

Converts an existing column to be an auto-increment column.

Sample

<addAutoIncrement tableName="person" columnName="id" columnDataType="int"/>

Available Attributes

tableNameName of the table to add the auto-increment to [required]
columnNameName of column to make auto-increment [required]
columnDataTypeCurrent data type of the column to make auto-increment [required]
