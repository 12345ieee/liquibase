"Add Lookup Table" Refactoring

Creates a lookup table containing values stored in a column and creates a foreign key to the new table.

Sample

<addLookupTable
    existingTableName="address" existingColumnName="state"
    newTableName="state" newColumnName="abbreviation"
    constraintName="fk_address_state"
/>

Available Attributes

existingTableNameName of the table containing the data to extract [required]
existingColumnNameName of the column containing the data to extract [required]
newTableNameName of lookup table to create [required]
newColumnNameName of the column in the new table to create [required]
constraintNameName of the foreign-key constraint to create between the existing table and the lookup table [required]
