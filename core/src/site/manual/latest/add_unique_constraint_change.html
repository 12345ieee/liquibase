"Add Unique Constraint" Refactoring

Adds a unique constrant to an existing column or set of columns.

<addUniqueConstraint tableName="person"
    columnName="id"
    constraintName="pk_person"/>

Available Attributes

tableNameName of the table to create the unique constraint on [required]
columnNamesName of the column(s) (comma separated if mulitiple) to create the unique constraint on [required]
constraintNameName of unique constraint [required]
tablespace"Tablespace" (file group in SQL Server) to create the index in [optional]
