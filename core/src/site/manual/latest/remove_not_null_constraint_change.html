"Drop Not-Null Constraint"

Makes a column null-able.

Sample

<dropNotNullConstraint tableName="employee" columnName="employer_id"/>

Available Attributes

tableNameName of the table containing that the column to drop the constraint from [required]
columnNameName of the column to drop the constraint from [required]
columnDataTypeCurrent data type of the column [required for MySQL & MS-SQL]
