"Execute Shell Command" Refactoring

Executes a system command. Because this refactoring doesn't generate SQL like most, using LiquiBase commands such as migrateSQL may not work as expected. Therefore, if at all possible use refactorings that generate SQL.

Sample

<executeCommand executable="mysqldump" os="Windows XP">
    <arg value="--add-drop-database"/>
    <arg value="--compress"/>
    <<arg value="dbName"/>
</executeCommand>

Available Attributes

executableName of the executable to run [required]
osList of operating systems on which to execute the command (taken from the os.name Java system property) [optional]

Available Sub-Tags

argArgument (in the "value" attribute) to pass to the executable.
