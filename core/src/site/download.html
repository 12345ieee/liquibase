Download LiquiBase Core

Requirements

LiquiBase has been designed for Java 1.5 and greater. There is a package built for 1.4, but it has not been well tested.

Installation

Extract liquibase.zip. In it you will find a liquibase-VERSION.jar This jar is all that is needed to run the migration tool from within Ant, Maven, Spring, or a servlet container. You can run the command line version of the migrator with the same jar file by simply using java -jar liquibase-VERSION.jar.

Download LiquiBase IDE

The current version of the stand-alone LiquiBase IDE is 0.6.

Download Stand-Alone IDE

Install LiquiBase Eclipse Plug-In

The current version of the LiquiBase Eclipse plug-in is 0.6. To install, add http://www.liquibase.org/update/eclipse/plugin to your Eclipse update sites.
