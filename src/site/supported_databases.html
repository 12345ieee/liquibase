Supported Databases

Due to variations in data types and SQL syntax, the following databases are currently supported:

MySQLNo Issues
PostgreSQL8.2+ is required to use the "drop all database objects" functionality.
OracleNo Issues
MS-SQLNo Issues

Additional database support will be a highlight of future releases.

If you have problems with a particular tag on a particular database system or want to suggest support for a currently unsupported database, please report the problem

If you find a problem with the generated SQL for your database, you can always use the <sql> tag to code whatever change you need to make.
