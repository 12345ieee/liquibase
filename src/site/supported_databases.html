Supported Databases

Unlike other database change tracking tools, LiquiBase will work with any JDBC-compliant database.

However, the refactoring tags (such as createTable) may need to generate different SQL for different database systems and have been primarily tested with Oracle, MySQL, PostgreSQL, and MS-SQL. Improved database support will be a highlight of the pre-1.0 release process.

If you have problems with a particular tag on a particular database system, please report the problem

If you find a problem with the generated SQL for your database, or if you are using an unsupported database, you can always use the <sql> tag to code whatever change you need to make, regardless of the DBMS.

Database Notes

Postgres

Postgres 8.2 is required to use the "drop all database objects" functionality.
