LiquiBase Development

Like all open-source projects, we encourage looking at the code, suggesting ideas, and even implementing fixes and improvements.

The LiquiBase code is hosted on SourceForge's SVN repository can be browsed online via ViewVC or checked out through SVN with the following command:

svn co https://liquibase.svn.sourceforge.net/svnroot/liquibase liquibase

If you are interested in contributing to LiquiBase, please contact us via the mailing list.
