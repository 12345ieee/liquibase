@NAV.MANUAL@

"Add Default Value" Refactoring

Adds a default value to the database definition for the specified column.

Sample

<addDefaultValue tableName="file"
    columnName="fileName"
    defaultValue="New File"/>

Available Attributes

tableNameName of the table to containing the column [required]
columnNameName of column to add a default value to [required]
defaultValueName of column to add a default value to [required]
