"Modify Column" Refactoring

Makes changes to an existing column such as the data type.

Sample

<modifyColumn tableName="person">
    <column name="firstname" type="varchar(5000)"/>
</modifyColumn>

Available Attributes

tableNameName of the table that the column to modify is in

Available Sub-Tags

columnNew column definition See the column tag documentation for more information
