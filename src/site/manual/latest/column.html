<column> tag

The "column" tag is a tag that is re-used throughout the LiquiBase XML when column definitions and column information is needed. As a result, not all the attributes of column make sense in each context it is used in.

Available Attributes

nameName of the column
typeData type of the column
valueValue to set the column to
defaultValueDefault value for column
autoIncrementIs column an auto-increment column

Available Sub-Tags

constraintsConstraint definitions

<constraints> tag

The "constraints" tag contains information about constraints on the column

Available Attributes

nullableIs column nullable?
primaryKeyIs column a primary key?
uniqueShould a unique clause be applied
referencesForeign key definition
foreignKeyNameForeign key name
deleteCascadeSet delete cascade
deferrableAre constraints deferrable
initiallyDeferredAre constraints initially deferred
