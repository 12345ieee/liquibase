<addNotNullConstraint> tag

Adds a not-null constraint to an existing table. If a defaultNullValue attribute is passed, all null values for the column will be updated to the passed value before the constraint is applied. 

<addNotNullConstraint
        tableName="employee"
        columnName="employer_id"
        defaultNullValue="1" />

Available attributes

tableNameName of the table to add the constraint to [required]
columnNameName of the column to add the constraint to [required]
defaultNullValueValue to set all currently null values to. If not set, change will fail if null values exist [optional]
