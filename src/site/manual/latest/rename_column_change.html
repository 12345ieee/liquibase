Rename Column"

Renames an existing column

Sample

<renameColumn tableName="person"
    oldColumnName="fname" newColumnName="firstName"/>

Available Attributes

tableNameName of the table containing that the column to rename [required]
oldColumnNameExisting name of the column to rename [required]
newColumnNameName to rename the column to [required]
columnDataTypeCurrent data type of the column (MySQL only)
