"Merge Columns" Refactoring

Concatenates the values in two columns, joins them by with string, and stores the resulting value in a new column.

Sample

<mergeColumns tableName="person"
    column1Name="phoneAreaCode"
    joinString="-"
    column2Name="phoneSuffix"
    finalColumnName="phone
    finalColumnType="varchar(50)"
/>

Available Attributes

tableNameName table containing the columns to join [required]
column1NameName column containing the first half of the data [required]
joinStringString to place include between the values from column1 and column2 (may be empty) [required]
column2NameName column containing the second half of the data [required]
finalColumnNameName of the column to create [required]
finalColumnTypeDatatype of the column to create [required]
